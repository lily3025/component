
## 使用说明

这里提供的 Zepto 模块，已经包含了 Zepto 的下列扩展：

- touch.js
- fx.js

### 外链形式

```html
<script src="{{src}}"></script>
```

### 线上外链

包含：Zepto 1.1.2，Zepto.touch.js, Zepto.fx.js

```html
<script src="http://s5.qhimg.com/static/30a0e35d13625672/zepto112.js"></script>
```

### 模块化加载

`qiwoo` 约定：Zepto 库只能以外链形式引入。


## 使用说明

这里提供的 Zepto 模块，已经包含了 Zepto 的下列扩展：

- touch.js

### 外链形式

```html
<script src="{{src}}"></script>
```

### 线上外链

包含：Zepto1.0，Zepto.touch.js

```html
<script src="http://s6.qhimg.com/static/64cc919e84aad780/zepto100.js"></script>
```

### 模块化加载

`qiwoo` 约定：Zepto 库只能以外链形式引入。
